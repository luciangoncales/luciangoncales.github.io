Material Suplementar - IHC 2020


Contextualização: Dados psicofisiológicos dos usuários, tais como as ondas cerebrais já foram utilizadas com técnicas de aprendizado de máquina, para identificar o nível de conhecimento, e de dificuldade dos desenvolvedores de software. Problema: Porém, pouco se sabe sobre o desempenho das técnicas de aprendizado de máquina para classificar a compreensão de código baseado em ondas cerebrais dos desenvolvedores. Objetivo: Este estudo avalia o desempenho de técnicas de aprendizado de máquina em conjunto com dados psicofisiológicos para classificar situações de compreensão de códigos por desenvolvedores de software. Método: Um aparelho de EEG wireless foi utilizado para coletar ondas cerebrais de desenvolvedores de software enquanto executavam tarefas de compreensão de código. Ao todo, cada um dos 35 participantes executou 10 tarefas de compreensão. A partir desse conjunto de dados gerado, as técnicas de Rede Neural (NN), Support Vector Machine (SVM), Naïve Bayes (NB) e Random Forrest (RF) foram treinadas. A eficácia dessas técnicas foi analisada através das métricas de acurácia, precisão e recall. Resultados: Entre as técnicas de aprendizado de máquina, o classificador de rede neural de quatro camadas, treinado com dados de EEG e reduzidos com Principal Component Analysis (PCA), obteve uma precisão de 84% para classificar situações de compreensão de código dos desenvolvedores de software. Conclusão: Os resultados apresentam o potencial das técnicas de aprendizado de máquina em classificar a compreensão de código baseados nos dados EEG dos desenvolvedores.
